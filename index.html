Hackful makes it insanely fun to read and contribute to Hackful Europe

Available on the iPhone App Store

  • Features
  • About & Credits

  • All recent news

    Start reading all recent posts and comments in a very simple and convinent way.

  • Swipe gesture

    Just swipe the entry you like and start sharing, commenting and up voting.

  • Comment & Up vote

    See how much up votes or replies a post or comment got. You can also up vote entries, comment posts and reply to comments with your account.

  • Open Source

    The whole project is open source and free. You can find the project on GitHub and start contributing today.
